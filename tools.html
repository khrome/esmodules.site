Tools

Currently there are a lot of rough edges and missing pieces that you may encounter some sharp edges. These are the libraries those of us who have come before have built to make our lives easier.

  • @environment-safe - A suite of browser and node compatible abstractions for use writing logic that runs everywhere without build.
  • moka - A mocha test runner that supports importmap generation, browser execution and more.
  • create importmap - A webservice to generate import maps
  • wing-kong - A javascript library and CL npm to statically generate importmaps
  • importly - A commandline typescript app to generate an importmap
